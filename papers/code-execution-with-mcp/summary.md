# 核心内容概要 (120字)

AI Agent 工具规模化面临 Token 成本与效率瓶颈。传统工具调用因上下文臃肿和数据冗余而低效。一种新的范式是“代码执行模式”，让 Agent 生成代码来调用工具。此模式通过按需加载、本地数据处理和更优的控制流，显著降低 Token 消耗、提升效率和隐私保护，是构建大规模自主 Agent 的关键。
