# 项目简报：Paper-Notes

## 1. 项目核心目标

`Paper-Notes` 是一个用于管理和处理学术论文的自动化工作流系统。其主要目标是将复杂的学术论文转化为易于理解、消化和分享的内容格式。

## 2. 核心功能

- **论文管理**：在结构化的目录中存储和组织论文（例如 PDF、原始文本）。
- **内容提取**：从论文中提取关键信息，如文本、图片和参考文献。
- **格式转换**：将论文内容转换为 Markdown、HTML 等标准格式。
- **内容摘要与发布**：生成内容的摘要（例如单句总结、要点），并将其发布到社交媒体平台（如 Twitter）。
- **自动化工作流**：通过脚本和命令行工具，将上述步骤串联成自动化的流程。

## 3. 项目范围

- **输入**：学术论文（主要是 PDF 格式）和其他相关资料。
- **处理**：解析、转换、摘要和格式化。
- **输出**：Markdown 文件、HTML 文件、图片以及准备好发布的社交媒体内容。
- **工具**：主要使用 Python 脚本和命令行工具来执行任务。

## 4. 技术栈与模式

- **编程语言**：Python
- **核心工具**：`requests`, `python-dotenv`, `argparse`, `pathlib`
- **版本控制**：Git，托管在 GitHub 上。
- **工作流模式**：通过命令行接口（CLI）触发的脚本化任务。
